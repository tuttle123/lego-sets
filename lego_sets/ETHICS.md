# Ethical Considerations

## Overview

This document covers the ethical aspects of using this LEGO set scraper.

## Key Points

1. **Respect Terms of Service**: Check the LEGO website’s terms to ensure scraping is allowed.
2. **Minimize Impact**: The script makes a single request per run to avoid overloading the server.
3. **Data Privacy**: Only public data is collected—no personal or sensitive information.
4. **Responsible Use**: Use the data responsibly and don’t harm the website or its users.

## Conclusion

This project follows ethical guidelines to ensure fair and respectful use of web scraping.
